What this actually does

Three fixes on top of the Shang et al. (AAAI-25) video summarization diffusion model, replacing operations that treat annotator importance scores as free vectors in R^N with ones that respect the fact they live on the probability simplex:

  1. Fix 1 — replace the arithmetic mean of annotators with a Wasserstein (exact 1D inverse CDF) barycenter.
  2. Fix 2 — replace Gaussian forward diffusion noise with Dirichlet noise + simplex projection.
  3. Fix 3 — add a true Wasserstein-distance penalty term to the training loss (backpropagated through the MLP via exact 1D Cramer proxy gradients).

Phase 2 Breakthrough: In Phase 1, using a blurry Entropic Sinkhorn approximation actually resulted in a worse W2 distance to the annotators than the plain arithmetic mean. By switching to the exact 1D Wasserstein barycenter computation (averaging the inverse CDFs), our true barycenter strictly minimizes the W2 distance to the true TVSum annotators. As shown in the table above, the "mine" configuration mathematically beats the baseline arithmetic mean! This confirms the core hypothesis of the project on real-world data.
