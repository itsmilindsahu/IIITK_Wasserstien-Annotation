Phase 1 Prototype→Algorithms

This proof-of-concept implements the three fundamental fixes on top of the Shang et al. (AAAI-25) video summarization diffusion model. It focuses on replacing operations that treat annotator importance scores as free vectors in RN with ones that explicitly respect the probability simplex structure.

The Methodology Pipeline

Instead of relying on standard unconstrained statistics, we introduce a topologically sound approach to handle annotation scores:

  • Fix 1 — Replace the arithmetic mean of annotators with a Wasserstein (Sinkhorn) barycenter.
  • Fix 2 — Replace Gaussian forward diffusion noise with Dirichlet noise + simplex projection.
  • Fix 3 — Add a Wasserstein-distance penalty term to the training loss.

Fix 1: Sinkhorn Barycenter

Using the arithmetic mean on probability distributions can create unnatural bimodal distributions that don't represent any individual annotator's intent. Instead, we use the Entropic Sinkhorn Barycenter to find the true geometric mean on the simplex.

def wasserstein_barycenter(annotator_scores, n_iter=40, reg=0.05):
    """K x N annotator score matrix -> 1 x N barycenter on the simplex."""
    K, N = annotator_scores.shape
    C = cost_matrix(N)
    Kmat = np.exp(-C / reg)

    u = np.ones((K, N))
    bary = np.ones(N) / N

    for it in range(n_iter):
        v = annotator_scores / (Kmat.T @ u.T).T.clip(1e-16)
        Ku = (Kmat @ v.T).T
        bary = np.exp(np.mean(np.log(Ku * u + 1e-16), axis=0))
        bary = sinkhorn_projection(bary, n_iter=1)
        u = bary[None, :] / Ku.clip(1e-16)

    return bary

Fix 2: Dirichlet Forward Process

Standard diffusion adds Gaussian noise, which forces the state off the probability simplex (creating negative probabilities and sums ≠ 1). We fix this by injecting Dirichlet noise and projecting back to the simplex using a Sinkhorn projection.

def forward_step_mine(x_star, t, T=200):
    ab = alpha_bar(t, T)
    delta = dirichlet_noise(len(x_star))
    
    # Apply diffusion schedule
    xt = np.sqrt(ab) * x_star + np.sqrt(1 - ab) * delta
    
    # Proj_delta step to return to the simplex
    xt = sinkhorn_projection(xt, n_iter=3)  
    return xt

Open Question for Phase 2:
On this synthetic two-camp video, the arithmetic mean stays bimodal while the barycenter smooths toward a single interpolated mode. However, in this run the barycenter's average W2 distance to the annotators came out higher than the arithmetic mean's under the simplified CDF-based W2 proxy. This is flagged to be resolved in Phase 2, as the 1D "sum of squared CDF differences" proxy isn't exactly equivalent to the true W2 the Sinkhorn barycenter minimizes.
