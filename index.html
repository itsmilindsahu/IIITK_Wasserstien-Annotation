Wasserstein Diffusion on the Annotation Simplex

A small, hand-built experiment on geometric aggregation for annotator disagreement.

This project began with a simple question: what if annotator scores were treated as objects on a probability simplex rather than as ordinary vectors? The answer is that the geometry changes the behaviour of the model, and that change is worth writing down.

I wanted this page to feel less like a static report and more like a research notebook: a place for the core idea, a few implementation details, and the evidence that came out of the first run.

The core idea

The original pipeline aggregated annotator scores in a way that ignored the simplex structure. That felt mathematically awkward, so I tested three small but meaningful changes that make the model respect the geometry of the problem.

1. Replace the mean

I swapped the arithmetic mean for a Wasserstein-Sinkhorn barycenter, so the aggregate is no longer just a coordinatewise average.

2. Replace Gaussian noise

The forward diffusion step now uses Dirichlet-style noise and a simplex projection, which is more faithful to the domain of the annotations.

3. Penalize with geometry

I added a Wasserstein penalty to the loss so the predictor is not only fitting pointwise targets, but also respecting distributional structure.

Notebook excerpt

Here is the small implementation sketch that captures the spirit of the experiment.

def training_step(pred, target, annotators, lam=0.1):
    mse = np.mean((pred - target) ** 2)
    w2 = w2_1d(pred, annotators)
    return mse + lam * w2

The loss is deliberately simple: keep the supervised signal, but add a regularizer that nudges the model toward geometric consistency.

noise = np.random.dirichlet(np.ones(n))
noisy_state = project_simplex(x_prev + noise)

On this synthetic two-camp example, the arithmetic mean stays visibly bimodal while the barycenter smooths toward a single interpolated mode. That is the qualitative behaviour I was hoping to see. The numerical outcome is still open to interpretation, and that is exactly why the project feels interesting enough to continue.

What remains open

The current results are encouraging, but they also reveal that the proxy metric is not yet fully aligned with the geometry the Sinkhorn barycenter is actually optimizing. That is the next thing I would tighten up before treating Phase 2 as a serious benchmark.
